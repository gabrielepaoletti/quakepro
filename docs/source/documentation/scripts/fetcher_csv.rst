Download waveforms from CSV
===========================

.. program:: qpf-csv

.. code-block:: bash
    qpf-csv <subcommand> [options]

.. option:: -p, --provider <provider_code>
   :type: string
   :required: yes
   :options: "IRIS", "INGV", "SCEDC"

.. option:: -n, --network <network_code>
   :type: string
   :required: yes
