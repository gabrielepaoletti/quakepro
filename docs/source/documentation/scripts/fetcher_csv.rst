Download waveforms from CSV
===========================

.. option:: qpf-csv

    Download seismic waveforms using FDSN web services.

    :Parameter

    :Parameter:
        **slice_obj** : *str*
            The slice object or 'random' to select a waveform. Default is 'random' 

    :parameter slice_obj: Union[slice, int, str], optional
        The slice object or 'random' to select a waveform. Default is 'random'
